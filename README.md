## 교통사고 통계 기반 사상자 비율 분석 및 예측 모델

**연령대·차종·차량용도별 교통사고 통계(2005~2024)**를 통합하고, 연도별 사상자 비율(사망·부상 / 사고 건수) 변화를 분석하며 기계학습 모델을 활용해 2024년 사상자비율을 예측하는 프로젝트입니다.

1. 프로젝트 개요
● 원시 데이터

국토부·경찰청 교통사고 통계(엑셀)

구성: 연령 / 차종 / 차량용도

범위: 2005–2024

● 데이터 특징

연도 구간별로 파일 분리 (0509, 1014, 1519, 2024)

시트·컬럼 구조가 다름 → 공통 구조로 변환 필요

● 분석 목표

시계열 기반 사상자비율(사망+부상 / 사고수) 계산

연령대·차종·차량용도별 위험도 분석

머신러닝 모델을 통한 2024년 사상자비율 예측

Linear Regression / RandomForest 비교

2. 데이터 로딩 및 전처리
2.1 데이터 구성

Key 종류

Key	설명
연령	가해운전자 연령대
차종	가해운전자 차종
용도	차량 용도(대·중·소분류)
연도 구간 파일	0509(2005–2009), 1014(2010–2014), 1519(2015–2019), 2024(2020–2024)
2.2 전처리 단계 요약

key × 연도 조합으로 모든 엑셀 파일 반복 로딩

불필요한 합계 컬럼 제거

NA 행 제거

key별로 컬럼 명 표준화

연령: 가해운전자 연령대 → 연령대

차종: 가해운전자 차종 → 차종

용도: 대분류/중분류/소분류 → 하나의 문자열(차량용도_full)로 통합

melt + pivot_table로 모든 파일 구조를 연도 기준 시계열로 통일

사망률, 부상률, 사상자비율 계산

key별 DF → 리스트로 저장 후 concat하여 df_all 생성

3. 통합 테이블(df_all) 구조
