교통사고 통계 기반 사상자 비율 분석 및 예측 모델

본 프로젝트는 **연령대·차종·차량용도별 교통사고 통계(2005~2024)**를 통합하고,
연도별 사상자 비율(사망·부상 / 사고 건수) 변화를 분석하며
기계학습 모델을 활용해 2024년 사상자비율을 예측하는 프로젝트입니다.

1. 프로젝트 개요

원시 데이터: 국토부/경찰청 교통사고 통계 엑셀 (연령, 차종, 용도 / 2005~2024)

데이터 형태가 시트별·구조별로 다름 → 공통 구조로 변환 필요

melt + pivot_table을 활용해 연령대 / 차종 / 용도 × 연도의 시계열 구조 생성

사망률, 부상률 → 최종적으로 사상자비율 계산

원-핫 인코딩 + RandomForest / LinearRegression 모델 학습

최종 목표: 2024년 사상자비율 예측 및 비교
